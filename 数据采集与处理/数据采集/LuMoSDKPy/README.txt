LuMoSDK 接收FZMotion动捕系统数据
包括：散点数据，刚体数据，人体数据，点集数据(只限在FZMotion2.0传输时接收)
注：该示例程序支持python3.*以上32位和64位，但是运行时需要安装ZMQ和Protobuf_3.20.3版本库
安装方式：在终端运行命令：pip install zmq 以及 pip install protobuf==3.20.3
安装成功后，在VS配置python环境即可使用
