# 指尖位置点击检测与可视化工具

## 📌 功能说明

本工具基于**模型推理得到的指尖位置序列**，对其在 **z 轴方向的位移变化**进行处理，用于识别鼠标点击等微小动作事件，并进行可视化。

---

## 📂 输入数据

输入为模型推理输出的指尖位置序列：

* 文件：`pred_tip_pose.txt`
* 格式：

```
frame_idx, x, y, z, r6_1, r6_2, ..., r6_6
```

其中：

* `(x, y, z)` 为指尖三维位置（单位 mm）

---

## 🚀 使用流程

### 1️⃣ 检测点击事件

运行：

```bash
python detect_clicks_from_tip4_consensus.py
```

👉 输入：

* 指尖位置数据

👉 输出：

* `click_events.csv`：检测到的点击事件
* `click_debug.csv`：中间信号（用于分析）

---

### 2️⃣ 曲线可视化（查看点击信号）

运行：

```bash
python 曲线播放 进度条.py
```

👉 功能：

* 显示 z 轴处理后的信号曲线
* 实时播放
* 点击触发时显示：

```
CLICK!
```

👉 支持：

* 播放 / 暂停
* 拖动进度条
* 单帧查看
* 自动淡化提示

---

### 3️⃣ 多信号分析（可选）

运行：

```bash
python 曲线分析.py
```

👉 功能：

* 对比不同信号：

  * 指尖 z 位移
  * 相对位移
  * 去趋势信号
  * 点击检测结果

👉 用于调参和验证检测效果

---

### 4️⃣ 三维可视化 + 点击标记（可选）

运行：

```bash
python viewer_with_click_events.py
```

👉 功能：

* 显示模型预测的指尖位姿
* 高亮显示检测到的点击事件
* 点击位置随时间淡化

---

## ⚙️ 关键参数（可调）

在检测脚本中：

```python
threshold = 7.0          # 点击触发阈值
recover_drop = 1.0       # 回弹幅度
min_active_points = 2    # 同时参与的点数
refractory_frames = 22   # 不应期（避免重复检测）
```

---

## 📌 注意事项

* 坐标需为统一坐标系（建议 mm）
* 点击检测基于 **z 轴下压 → 回弹** 的模式
* 参数需根据数据适当调整

---

## 📊 输出说明

| 文件               | 含义        |
| ---------------- | --------- |
| click_events.csv | 每次检测到的点击帧 |
| click_debug.csv  | 中间计算信号    |
